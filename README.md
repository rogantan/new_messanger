# new_messanger
## Общее назначение разработки 
Разработать новый мессенджер в связи с импортозамещением иностранных мессенджерей для общения с зарегистрированными пользователями без подтверждения личности
## Функциональное назначение
* Регистрация по логину и паролю
* Авторизация по логину и паролю
* Поиск пользователей мессенджера по логину
* Написание сообщений между пользователями
* Создание групп для одного и болле пользователей
* Написание сообщений в группе
* Сохранение истории сообщений в чатах
## Нефункциональные требования
* Шифрование паролей
* Защита от SQL-инъекций
## Разделение на микросервисы
* Сервис авторизации и регистрации
* Сервис чатов (общения)
* Клиентская часть
  ** Веб-приложение
  ** Мобильное приложение
  ** Десктопное приложение (в планах)
